Parameters

  • path (str, optional) – the current path used in the file dialog, defaults to “.”

  • caption (str, optional) – the title of the dialog, defaults to “File Dialog”

  • filter (str, optional) –

    List of filters separated with “;;”, defaults to “All Files (*)”.

    Example:

    filter="Images (*.png *.xpm *.jpg);;Text files (*.txt);;XML files (*.xml)"

  • fileMode (FileMode, optional) – The file mode defines the number and type of items that the user is expected to select in the dialog, defaults to Anyfile

  • acceptMode (AcceptMode, optional) – TThe action mode defines whether the dialog is for opening or saving files, defaults to AcceptOpen
