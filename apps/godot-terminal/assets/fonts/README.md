# Fonts Folder

This folder contains all font files required by the Godot Terminal app.

- Place any necessary font assets here, such as Nerd Fonts or custom typefaces.
- For best results, add a Nerd Font (e.g., JetBrainsMonoNerdFontMono-Regular.ttf) as described in the main README.
- The app will use these fonts for terminal rendering and will fall back to the default Godot font if none are present.

**Note:** Running `setup.sh` in the main app folder will automatically download the recommended Nerd Font into this directory if it is not already present.
